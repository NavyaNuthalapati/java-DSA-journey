## GFG's problem on array's!

Q1.Given an array arr[], check whether it is sorted in non-decreasing order. Return true if it is sorted otherwise false.
Check it out [here](https://www.geeksforgeeks.org/problems/check-if-an-array-is-sorted0701/1)

## Insightfull 
- For a sorted array max element is present in arraylength-1
- If array size is 0 then we must return true (which is good practice for some test cases)
- We can use nested loops to fin it out !
